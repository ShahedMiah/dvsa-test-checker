DVSA Test Booking Guide

What You'll Need:

  • UK Driving License Number
  • Theory Test Pass Number OR Existing Booking Reference
  • Preferred Test Centre Location

Steps to Check Test Availability:

1. Visit the DVSA Website

Go to the official DVSA booking site:

DVSA Booking Portal â†’

2. Login Details

Enter your driving license number and either:

  • Theory Test Pass Number
  • Or Booking Reference (if rebooking)

3. Complete Security Check

Complete any CAPTCHA or security verification if prompted.

4. Search for Test Centres

After logging in:

  • Click on 'Change Test Centre'
  • Enter your preferred location or postcode
  • Use the 'Show More' button to see additional centres

5. Check Available Dates

Select a test centre to view available dates and times.

Tips for Finding Tests:

  • Check early morning or late at night for new slots
  • Consider checking multiple test centres in your area
  • Be prepared to book quickly as slots can fill up fast
  • Check regularly as cancellations can occur at any time

Important Notes:

  • Only use the official DVSA website
  • Avoid using automated tools as they may get you blocked
  • Make sure you're eligible before booking
  • Have your payment method ready
